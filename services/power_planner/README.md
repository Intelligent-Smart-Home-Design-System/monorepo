# power_planner

`power_planner` - это C++ модуль для расчета питания устройств умного дома.

## Что делает модуль

Модуль загружает входные данные из JSON и проверяет, можно ли запитать выбранные устройства от доступных источников питания.

Сейчас реализовано:

- загрузка устройств, источников питания, подключений и параметров расчета из JSON
- автоматическое распределение устройств с питанием от сети по подходящим источникам
- расчет номинальной и пиковой нагрузки для каждого источника питания
- поиск перегруженных источников и устройств без питания
- расчет сроков замены батареек и стоимости обслуживания
- вывод результата расчета в JSON.

## Входные данные

Пример входных данных находится в файле:

```text
data/sample_input.json
```

Во входных данных есть:

- `devices` - устройства умного дома с параметрами питания и батареек;
- `power_sources` - источники питания с напряжением и ограничением по току;
- `connections` - уже заданные подключения устройств к источникам питания;
- `options` - период расчета и текущая дата.
