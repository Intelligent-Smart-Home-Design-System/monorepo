# Модуль расстановки устройств

## Работа правил для устройств
Для устройств определяется интерфейс `Rule`, главным методом которого является `Apply`
```go
Apply(apartment *entities.Apartment) map[string]map[string]*entities.Placement
```
который по плану квартиры выдает зависимость `roomID -> deviceID -> devicePlacement` (чуть более подробно в коде программы).

Каждое устройство будет реализовывать интерфейс `Rule`, что позволит расставить их во всей квартире (в нужных местах, в нужном количестве) с помощью своего метода `Apply`.

## Запуск тестов
Для запуска тестов необходимо перейти из корневой директории в директорию layout и запустить тесты:
```bash
cd services/layout
go test -v ./tests/rules/...
```
